In Step 1 (Project Registry) of the Quotation Builder, you can select an existing client or project from the dropdown menus to pre-fill details. To register a new one, select 'Create New Client' or 'Create New Project' and enter the name, phone, email, site address, and estimated budget to proceed.

In Step 2 (Room Allocation) of the Quotation Builder, you can define your project's layout by adding spaces (like Living Room, Kitchen, Bedrooms, etc.). You can search and select from a predefined list of standard spaces in the right sidebar or type in a custom name to add custom zones instantly.

In Step 3 (Catalog Item Mapping), select the active room tab on the left, then browse the Master Catalog in the right sidebar. Click any item to add it. Once added, you can customize the brand, quantity, markup percentage, discount, and pricing details directly inside the item card.

SpaceIO supports three flexible pricing modes for items, which you can toggle dynamically:
• Piece: Fixed pricing per unit (quantity × unit price).
• Sq Ft (Square Feet): Auto-calculates cost based on dimensions (Length × Height/Breadth) multiplied by a base rate per sq ft.
• Running Ft (Running Feet): Calculates cost based on Length multiplied by a base rate per running foot.
